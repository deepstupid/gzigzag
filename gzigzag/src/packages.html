GZigZag src/ implementation Java packages.

This html document contains the overview documentation and the package-specific documentation of the GZigZag src/ implementation.

This file is not meant to be read by itself (even though it is valid html) --- it is meant to be split apart by scripts and moved to package.html and overview.html files.

Overview

This is the second GZigZag implementation (codenamed "src/").

org.gzigzag

This package defines the various core APIs that we will attempt to change only as slowly as possible.

org.gzigzag.benchmark

This package provides classes related to speed-testing.

org.gzigzag.client

This package contains classes which implement the GZigZag client frontend.

org.gzigzag.gfx

The low-level graphics code for replacing java.awt.Graphics with faster and more powerful primitives on some platforms.

org.gzigzag.impl

The classes that implement the core APIs in org.gzigzag.

org.gzigzag.impl.clasm

Implementation of CLASM, a low-level cellular language.

org.gzigzag.mediaserver

An implementation of a low-level basis for the Xanadu media model: bit blocks with truly unique global IDs.

org.gzigzag.mediaserver.ids

Code for managing the globally unique IDs in mediaservers.

org.gzigzag.mediaserver.http

An implementation of HTTP for allowing communication to/between mediaservers.

org.gzigzag.mediaserver.http.server

An implementation of HTTP server for allowing communication to/between mediaservers.

org.gzigzag.mediaserver.http.client

An implementation of HTTP client for allowing communication to/between mediaservers.

org.gzigzag.mediaserver.storage

The code that handles the actual storage of mediaserver blocks.

org.gzigzag.test

Classes which include the tests from the various packages into single classes that can be run under JUnit.

org.gzigzag.util

Code that the Java platform should provide, but doesn't ;).

org.gzigzag.vob

The high-level graphics library for animation and connections. Does not depend on the rest of gzigzag.

org.gzigzag.vob.demo

Various demos for vobs.
