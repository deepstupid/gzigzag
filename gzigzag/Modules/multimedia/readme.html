2. How to use macros

To define a macro:
  • Just use the MediaZOb you want to use ("mm.rect" below).

  • Replace some of the parameters with the word "param" :-)

  • To the cell next from "param" on "d.1" write the name of your parameter. ("mywidth" below)

  • Optional: To the cell next on "d.1" write a default value.

To use a macro:
  • Clone the name of the MediaZOb...

  • Give the clone some of the parameters you have used in your macro definition.
